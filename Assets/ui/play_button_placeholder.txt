PLACEHOLDER: Play Button Assets

These files are placeholders for the PLAY button states.

REQUIREMENTS:
- Size: 400x80px (@1x) or 800x160px (@2x)
- Format: PNG
- Background: Deep red (#6C1B1B) with subtle gradient
- Text: "PLAY" in large bold serif font (Cinzel/Trajan)
- Text Color: Bright gold (#D4AF37 or #B8860B)
- Border: Inner golden border (2-3px) with emboss effect
- Style: Slight 3D/emboss effect for depth

STATES NEEDED:
1. play_button_normal.png - Default state
2. play_button_hover.png - Hover state (slightly brighter, glow effect)
3. play_button_pressed.png - Pressed state (darker, pressed effect)

RECOMMENDED CREATION:
- Use image editing software (Photoshop, GIMP, etc.)
- Create base button with red background
- Add golden inner border
- Add "PLAY" text in serif font
- Apply emboss/shadow effects
- Export three states

REPLACEMENT:
Replace these .txt files with actual PNG files in the same directory.

USAGE IN XAML:
The button will use XAML styles instead of images for better scalability.
See Themes/Controls.xaml -> PlayButtonStyle
